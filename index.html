Download LaTeX

The first thing you'll need to do today is install LaTeX on your own computer. There are several different ways to get LaTeX, depending on what operating system you use.

Linux

The simplest way to install LaTeX under Linux is to use your distribution's package manager.

Ubuntu and Debian users can enter the following in a terminal:

$ sudo apt-get install texlive-latex-recommended

Windows

The easiest way to get LaTeX on Windows is to download and install MikTeX. MikTeX gives you a full LaTeX distribution along with a nice editor and all the required programs to use LaTeX. It also comes with its own package manager, which makes it easy to download and install new LaTeX addons.

You can download MikTeX here. Scroll down to "Basic MiKTeX 2.9" Installer and click on the Download button.

Mac OSX

The preferred way to obtain LaTeX for OSX appears to be to install MacTeX, which can be obtained here.

Download and Install RevTeX

You can download RevTeX from the APS website. If you're running Linux, you should download and install the file from there. Just download it, unzip it, and follow the instructions in the README.

On Windows, you can install RevTeX from MikTeX's package manager. The package you want to look for is simply called "revtex". It should take care of everything.

Under OSX, you should be able to simply follow the same procedure as for Linux.

Get the Template File

For this session, you will be using a template which we have prepared for you. It demonstrates many basic features that you will use all the time. After installing LaTeX, your next step should be to download and unpack the template.

latex_session_2011.tar.bz2 (for Linux)

latex_session_2011.zip (for Windows and OSX)

Edit the Template

Now, you'll need to open up the template and edit it. The best way to do this is to use a text editor. This is a program whose specialty is editing plain text files.

Under Linux, Gedit is a simple and relatively easy to use choice. If you're feeling adventurous, the text editor Emacs along with the AUCTeX environment is probably the most powerful and efficient way to edit LaTeX. Along with Emacs, the other "standard" text editor is Vim, for which Vim-LaTeX is available. Any of these editors can be installed through your package manager.

For Windows, you can use TexWorks, which comes with MikTeX. Another popular choice is Notepad++, which is a fully-featured text editor with lots of nice features.

If you're on OSX, MacTex comes with a few options, including TeXShop and TeXWorks. There is also an OSX port of Vim called macvim.

Useful Documentation

This LaTeX session can only teach you the basics. To really get down to using LaTeX, you'll need some good documentation to refer to. Fortunately, since LaTeX is so widely used, there are lots of good references to be found.

  • A good starting point is the official LaTeX Project web site which has links to many guides and also has a good list of books about LaTeX.
  • The Wikibooks LaTeX guide is another excellent and quite comprehensive resource which is loaded with information.
